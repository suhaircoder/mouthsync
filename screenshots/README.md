# Application screenshots

Place **UI screenshots** of MouthSync here (PNG or JPG recommended).

## Suggested filenames

| File | Content |
|------|---------|
| `01-main.png` | Main screen — portrait + audio upload, generate |
| `02-worker-settings.png` | Worker (RunPod) settings panel |
| `03-history.png` | Generation history |
| `04-result.png` | Result / video playback |
| `05-prep-settings.png` | Photo / audio prep options (if shown in UI) |

After adding images, reference them from the root [README.md](../README.md), for example:

```markdown
![Main screen](./screenshots/01-main.png)
```

## Tips

- Resolution: **1280–1920 px** wide is enough for GitHub README.
- Use **PNG** for crisp UI text; **JPG** for large photo previews.
- Do not commit secrets (API keys, Pod URLs with tokens) — crop or blur if needed.
